一个移动端react美食app

主代码分支：main

分支开发与合并：
pull request -- new pull request -- 选择分支 -- 填写合并信息 -- 提交

git branch: 查看分支列表
git branch new-branch-name: 新建分支
git branch -D delete-branch-name: 删除分支
git checkout chekout-branch-name: 检出分支
git checkout -b chekout-branch-name: 检出分支，若没有则新建再检出
